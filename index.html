conciliación cruzada · tickets vs instalaciones vs odoo
Lógica: Se cruzan clientes en Wishup "Pendiente" con historial de tickets. Si un cliente sin activar ya genera tickets de suspensión o calidad, es señal crítica. Para cruce exacto con Odoo se necesita el campo ID Suscripción en los reportes de tickets.

Hoja de ruta — cómo cerrar el loop
Paso 1 — Corto plazo
Agregar campo ID Suscripción Odoo al reporte de tickets en Odoo Helpdesk. Permite cruce exacto cliente-contrato.
Paso 2 — Mediano
Exportar campo Etapa / Estado Odoo en instalaciones Wishup para ver suspendidos vs pendientes de activar.
Paso 3 — Automatizado
Apps Script cruza hojas 22_CONCILIACION + 25_TICKETS y genera alerta cuando cliente activo acumula 3 tickets en 30 días.
